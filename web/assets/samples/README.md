# Sample posters

Three real exports from the map-poster app, used on the landing page (`web/index.html`).

- `sample-carrara.png` — Bình Thạnh, **Carrara** theme (cream/ink).
- `sample-noir.png` — Bình Thạnh, **Noir** theme (dark, high-contrast).
- `sample-song-sai-gon.png` — Bình Thạnh, **Sông Sài Gòn** theme (Vietnamese palette: warm sepia land, teal river).

All three are genuine 300 DPI A3 portrait exports from `poster.html`'s Export
step, downscaled to 1000px wide (keeping the A3 aspect ratio, ~1000×1414) to
keep the page weight reasonable. They are not mockups — same rendering
pipeline (boundary mask, category layers, ward borders, hero title
placement) a user gets from the app.

Regenerate anytime: open `poster.html`, pick an area, choose a theme, and use
Export to download a fresh print-resolution PNG. To refresh these specific
samples, export at A3 with the theme named above, downscale to ~1000px wide,
and overwrite the matching file here.
